# Curso de Docker Avanzado - Platzi

## 📌 Descripción del Proyecto

Este repositorio contiene las prácticas y el proyecto final desarrollados como parte del **Curso de Docker Avanzado** de Platzi. El objetivo principal de este proyecto fue aprender a fondo y aplicar técnicas de optimización, seguridad y despliegue de aplicaciones, utilizando prácticas avanzadas de contenedorización que simulan entornos reales y escalables.

## 🚀 Tecnologías y Conceptos Aplicados

A lo largo de este proyecto, se han implementado técnicas avanzadas de Docker, incluyendo:

- **Multi-stage Builds:** Optimización del proceso de construcción de imágenes para reducir drásticamente su tamaño y mejorar la seguridad, separando las dependencias de compilación del entorno de producción final.
- **Gestión de Capas (Layers):** Aprovechamiento eficiente de la caché de Docker para acelerar los tiempos de construcción y optimizar las actualizaciones de la aplicación.
- **Automatización CI/CD:** Creación de flujos de integración y despliegue continuo para automatizar pruebas y liberaciones.
- **Redes en Docker:** Configuración avanzada de redes para permitir la comunicación segura y eficiente entre múltiples servicios y contenedores.
- **Balanceo de Carga:** Distribución del tráfico de red entre múltiples instancias de la aplicación para garantizar alta disponibilidad y tolerancia a fallos.
- **Despliegue en la Nube:** Transición y despliegue eficiente de la infraestructura desde un entorno de desarrollo local hacia plataformas en la nube.

## 🎯 Enfoque de Aprendizaje

> **Nota sobre el Control de Versiones:**
> Este proyecto fue concebido y ejecutado con el objetivo exclusivo de dominar la arquitectura, optimización y orquestación de contenedores con **Docker**. Por lo tanto, el flujo de trabajo en Git se ha mantenido intencionalmente simple y lineal (sin un manejo complejo de ramas o *branching*). El enfoque de evaluación debe centrarse 100% en las configuraciones, los Dockerfiles y los archivos de composición (Docker Compose).
